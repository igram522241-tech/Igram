# iGram - Instagram Video Downloader

iGram is a free web-based tool that allows users to download publicly available Instagram Reels, videos, photos, stories, and IGTV content directly from their browser. It works on desktop and mobile devices without requiring software installation or account registration.

Website: https://igram.click/

## Features

- Download Instagram Reels
- Save public videos and photos
- Download Stories
- Fast browser-based tool
- No installation required
- Works across multiple devices

If you're looking for a simple way to save publicly available Instagram content for offline viewing, iGram provides a clean and user-friendly experience.
